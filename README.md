# Expense Tracker Web Application

A simple expense tracking web application built using AWS Serverless architecture.

---

## Project Description

The Expense Tracker allows users to:
- Add expenses with category, amount, and note
- View all expenses in a table
- Understand basic serverless web application flow

This project was created for learning AWS and full-stack development concepts.

---

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Java (AWS Lambda)

### AWS Services
- Amazon S3 (Static website hosting)
- AWS Lambda (Backend logic)
- Amazon API Gateway (REST API)
- Amazon DynamoDB (Data storage)
- Amazon CloudWatch (Logs & monitoring)

---

## Architecture

Frontend (S3)  
→ API Gateway  
→ Lambda Function  
→ DynamoDB
